# 水域安全小幫手

我們旨在幫助提高水域安全，預防水上事故，保護生命。

## 參賽官網

[大專校院資訊應用服務創新競賽](https://innoserve.tca.org.tw/)

## 項目介紹

**水域安全小幫手** 是一個提升水域安全的應用程序。它包括實時監控、預警系統和安全教育，以幫助用戶更好地了解和管理水域安全。

## 功能

- **實時監控**: 實時跟蹤水域狀況，包括天氣等重要信息。
- **預警系統**: 在檢測到潛在危險時，立即發出警報。
- **安全教育**: 提供水域安全相關的教育資源。
- **用戶反饋**: 收集用戶反饋，持續改進系統。
